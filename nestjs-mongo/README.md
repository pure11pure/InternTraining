### Start

```
npm i -g @nestjs/cli
# choose npm
nest new project-name
cd project-name
```

### run

```
npm start

# watch เวลาแก้ไข มันจะแก้ตาม
npm run start:dev
```

### create

```
# สร้าง Controller
nest generate controller <name>
# สร้าง Service
nest generate service <name>
# สร้าง Module
nest generate module <name>
```
